In this guide, we will demonstrate how to set up a new Next.js project using the custom `Stepper` component. Follow the steps below to get started.

## Preview

<Stepper>
  <StepperItem title="Step 1: Create a New Next.js App">
    Start by creating a new Next.js app using the `create-next-app` tool. Open
    your terminal and run the command: ```bash npx create-next-app@latest
    my-nextjs-app ``` Replace `my-nextjs-app` with your desired project name.
  </StepperItem>
  <StepperItem title="Step 2: Navigate to the Project Directory">
    Once the setup is complete, navigate to your new project directory. Run:
    ```bash cd my-nextjs-app ``` This will switch your terminal to the project
    folder.
  </StepperItem>
  <StepperItem title="Step 3: Start the Development Server">
    Start the Next.js development server by running the following command:
    ```bash npm run dev ``` Your application will now be available at
    `http://localhost:3000` in your browser.
  </StepperItem>
</Stepper>
